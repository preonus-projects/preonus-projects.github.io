# Korus Framework Tool

A complete **project management dashboard** and **control panel** for Korus Framework inside **Visual Studio Code**.

This extension streamlines your entire Korus development workflow — from project creation to lifecycle management, builds, testing, dependency analysis, and code generation insights — all inside VS Code.

## Key Highlights
- Manage Korus Java projects visually
- Start, stop, build, and test with one click
- Explore source code, components, and dependencies
- Track generated registry and component statistics
- Zero CLI required for daily development
