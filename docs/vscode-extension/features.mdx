# Features

## Project Management
- Auto-detect Korus projects
- Quick project switching
- Built-in project generator
- Real-time status updates

## Lifecycle Control
- Start / Stop / Restart projects
- Live status logs
- Port conflict detection
- Clean project artifacts

## Build & Compilation
- Quick Compile
- Clean Build
- Full Build
- Verify Build
- Package (JAR)
- Integrated terminal output

## Source Code Explorer
- Hierarchical Java package tree
- Create new classes and packages
- Rename / delete files
- Instant file opening

## Testing Integration
- Run all tests with one click
- Maven lifecycle test execution
- Terminal-based results

## Dependency Management
- Visual dependency tree
- Resolve dependencies
- Download sources
- Detect conflicts

## Project Information
- View project metadata
- Open pom.xml
- Edit application.properties
- View component statistics

## Code Generation Insights
- View generated registry files
- Count components by type
- Track project growth
