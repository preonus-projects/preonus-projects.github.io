Installation Guide

This guide walks you through installing Korus Framework in your project.

Prerequisites‚Äã

Before installing Korus, ensure you have:

  • Java 17+ or Kotlin 1.9+
  • Maven 3.6+ or Gradle 8.0+
  • An IDE (IntelliJ IDEA or VS Code recommended)
Java17+ Kotlin1.9+ Maven3.6+ Gradle8.0+

Quick Install‚Äã

Add Korus to your project by configuring the build system. Korus uses a Build Parent for Maven and a standard plugin-style approach for Gradle.

Project Configuration
  • Maven
  • Gradle
<project xmlns="http://maven.apache.org/POM/4.0.0">
  <modelVersion>4.0.0</modelVersion>

  <parent>
      <groupId>io.korus</groupId>
      <artifactId>korus-build-parent</artifactId>
      <version>1.0.0-SNAPSHOT</version>
  </parent>

  <artifactId>my-korus-app</artifactId>

  <dependencies>
      <dependency>
          <groupId>io.korus</groupId>
          <artifactId>core-starter</artifactId>
      </dependency>
      <dependency>
          <groupId>ch.qos.logback</groupId>
          <artifactId>logback-classic</artifactId>
      </dependency>
  </dependencies>

  <build>
      <plugins>
          <plugin>
              <groupId>org.apache.maven.plugins</groupId>
              <artifactId>maven-compiler-plugin</artifactId>
              <configuration>
                  <compilerArgs>
                      <arg>-Akorus.processing.verbose=true</arg>
                  </compilerArgs>
              </configuration>
          </plugin>
      </plugins>
  </build>
</project>
plugins {
  id 'java'
  id 'io.korus.plugin' version '1.0.0'
}

group = 'com.example'
version = '1.0.0-SNAPSHOT'

repositories {
  mavenCentral()
}

dependencies {
  implementation 'io.korus:core-starter'
  runtimeOnly 'ch.qos.logback:logback-classic'
  
  // Annotation Processor
  annotationProcessor 'io.korus:korus-processor'
}

tasks.withType(JavaCompile) {
  options.compilerArgs << "-Akorus.processing.verbose=true"
}

Verify Installation‚Äã

Create a simple service to verify the build-time wiring and code generation.

HelloService Verification
  • Java
  • Kotlin
package com.example.services;

import io.korus.core.annotation.Service;

@Service
public class HelloService {
  public String sayHello(String name) {
      return "Hello, " + name + "!";
  }
}
package com.example.services

import io.korus.core.annotation.Service

@Service
class HelloService {
  fun sayHello(name: String): String {
      return "Hello, $name!"
  }
}

Build and compile your project:

Build Command
  • Maven
  • Gradle
mvn clean compile
./gradlew classes

IDE Configuration‚Äã

IntelliJ IDEA‚Äã

  1. Enable Annotation Processing:
  • Settings ‚Üí Build, Execution, Deployment ‚Üí Compiler ‚Üí Annotation Processors
  • Check "Enable annotation processing"
  1. Kotlin Support: Ensure the KAPT or KSP plugin is enabled if using Kotlin.

VS Code‚Äã

Install the Korus VS Code extension for enhanced support:

code --install-extension korus.korus-initializer
