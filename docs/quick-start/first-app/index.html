Build Your First Korus Application

Let's build a simple REST API in under 5 minutes using Korus Framework!

What We'll Build‚Äã

A REST API with:

  • Health check endpoint
  • User management service
  • Dependency injection
  • Sub-200ms startup time

Step 1: Create Service Layer‚Äã

First, create a service to handle business logic:

UserService - Business Logic
  • Java
  • Kotlin
package com.example.services;

import io.korus.core.annotation.Service;
import java.util.List;
import java.util.ArrayList;

@Service
public class UserService {
  private final List<String> users = new ArrayList<>();
  
  public UserService() {
      // Initialize with sample data
      users.add("Alice");
      users.add("Bob");
  }
  
  public List<String> getAllUsers() {
      return new ArrayList<>(users);
  }
  
  public void addUser(String name) {
      users.add(name);
  }
  
  public String getUser(int index) {
      if (index >= 0 && index < users.size()) {
          return users.get(index);
      }
      return null;
  }
}
package com.example.services

import io.korus.core.annotation.Service

@Service
class UserService {
  private val users = mutableListOf("Alice", "Bob")
  
  fun getAllUsers(): List<String> = users.toList()
  
  fun addUser(name: String) {
      users.add(name)
  }
  
  fun getUser(index: Int): String? {
      return users.getOrNull(index)
  }
}

Step 2: Create REST Controller‚Äã

Now create a controller to expose REST endpoints:

UserController - REST API
  • Java
  • Kotlin
package com.example.controllers;

import com.example.services.UserService;
import io.korus.web.annotation.RestController;
import io.korus.web.annotation.GetMapping;
import io.korus.web.annotation.PostMapping;
import io.korus.web.annotation.PathVariable;
import io.korus.web.annotation.RequestBody;
import io.korus.web.exception.NotFoundException;

import java.util.List;

@RestController("/api")
public class UserController {
  private final UserService userService;

  public UserController(UserService userService) {
      this.userService = userService;
  }

  @GetMapping("/users")
  public List<String> getAllUsers() {
      return userService.getAllUsers();
  }
  
  @GetMapping("/users/{id}")
  public String getUser(@PathVariable int id) {
      String user = userService.getUser(id);
      if (user == null) {
          throw new NotFoundException("User not found");
      }
      return user;
  }
  
  @PostMapping("/users")
  public void addUser(@RequestBody String name) {
      userService.addUser(name);
  }
  
  @GetMapping("/health")
  public String health() {
      return "OK";
  }
}
package com.example.controllers

import com.example.services.UserService
import io.korus.web.annotation.RestController
import io.korus.web.annotation.GetMapping
import io.korus.web.annotation.PostMapping
import io.korus.web.annotation.PathVariable
import io.korus.web.annotation.RequestBody
import io.korus.web.exception.NotFoundException


@RestController("/api")
class UserController(
  private val userService: UserService
) {
  @GetMapping("/users")
  fun getAllUsers() = userService.getAllUsers()
  
  @GetMapping("/users/{id}")
  fun getUser(@PathVariable id: Int): String {
      return userService.getUser(id) 
          ?: throw NotFoundException("User not found")
  }
  
  @PostMapping("/users")
  fun addUser(@RequestBody name: String) {
      userService.addUser(name)
  }
  
  @GetMapping("/health")
  fun health() = "OK"
}

Step 3: Build and Run‚Äã

That's it! No need to create a main class. Korus auto-generates the entry point.

Build and Run
  • Maven
  • Gradle
mvn clean compile
mvn exec:java
./gradlew clean build
./gradlew run

Test Your API‚Äã

Your application is now running! Test the endpoints:

curl http://localhost:8080/api/health
curl http://localhost:8080/api/users
curl -X POST http://localhost:8080/api/users -H "Content-Type: application/json" -d "Charlie"

What Just Happened?‚Äã

During compilation, Korus:

  1. Scanned your @Service and @RestController annotations
  2. Generated dependency injection code
  3. Created route mappings
  4. Built a complete application entry point
  5. Eliminated all reflection overhead

You now have a working Korus application with ultra-fast startup and zero reflection overhead!
